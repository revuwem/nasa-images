## Overview

### Getting started

#### Run the development server

```bash
yarn dev
```
Open [http://localhost:5173/](http://localhost:5173/) with your browser to see the result.

#### Run tests

```bash
yarn test
```

To learn more about Vite go to [Vite Official Documentation](https://vitejs.dev/).
